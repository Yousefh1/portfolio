Library app

The library app is designed to record and save book-related data such as the author, page number, and read status. The app uses local storage to store the data, making it easily accessible for users. This feature eliminates the need for an external database. With this app, users can keep track of their reading progress and organize their book collection efficiently.This project was build with Javascript, CSS, HTML.
